## 总的

每一条消息内部如下
{"t":"Msg","c":"内容"}

### Msg 文本类型

（包括图文混排 回复）
内容看起来是这样
[["m", "早上好"],["e", "Loaded","lib/emoticon/news/s1.png",1],["m", "早上好"]]

| m    | 内容 |
| ---- | ---- |
| 文本 | Str  |

| emoji | 加载状况   | 路径     | 表情id |
| ----- | ---------- | -------- | ------ |
| 表情  | True/False | Str/None | Num    |

注：该表情是指腾讯的表情，若有emoji则需要字体支持，较为麻烦

| img  | 加载状况   | 路径     |
| ---- | ---------- | -------- |
| 图像 | True/False | Str/None |
